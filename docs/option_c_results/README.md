# Option C verification — results

Companion plots for [Python-Model-OT-Control issue #4](https://github.com/ajaytalati/Python-Model-OT-Control/issues/4) and the [model fix plan](https://github.com/ajaytalati/Python-Model-OT-Control/blob/main/version_1/docs/issue_evidence/v_h_inversion/model_fix_plan.md).

## `lambda_sweep.png`

Sweep over (λ, λ_Z) baselines, each at fixed V_n=0.3, V_c=0, T_0=0.5, deterministic, D=14d. Generated by `scripts/option_c_lambda_sweep.py`.

Key reading: at the spec's λ=32, the system is saturated and V_h has no effect (Δ=0.003 across V_h ∈ [0, 4]). At λ=4, λ_Z=1 the V_h response is monotonically increasing with Δ=0.74 — clinically meaningful anabolic response. Crosses the spec target T*=0.55 around V_h=0.7.

## `heatmap_compare.png`

V_h × V_n grid comparison: pre-fix vendored vs Option C at λ=4, λ_Z=1. Generated by `scripts/option_c_heatmap.py`.

- **Left (pre-fix)**: argmax at (V_h=0, V_n=0.5), T_end=0.519. V_h is anti-anabolic; the bright corner is the *wrong* clinical regime (no vitality intervention).
- **Right (Option C)**: argmax at (V_h=4, V_n=2), T_end=0.972. V_h is monotonically anabolic; the bright corner is the *clinically meaningful* regime (high vitality intervention).

The structural inversion is fixed.

## Test results

```
$ pytest tests/swat/                                        # vendored (pre-fix)
8 failed, 8 passed in 318.83s

$ pytest tests/swat/ --variant option-c \
                    --lambda-base 4 --lambda-z-base 1
4 failed, 12 passed in 145.25s
```

The 4 remaining failures under Option C are calibration drifts (T values slightly outside spec bands, sleep fraction high) — tuning issues, not structural. The structural-inversion tests (V_h-anabolicity, V_n-catabolicity, dose-response, no-backwards-optimum) all PASS.
